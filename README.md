## Prerequisites
```bash
python 3
```

## Getting started
```bash
# clone the project
git clone https://github.com/Joao-MCF/cannibal_missionary.git

# enter the cloned directory
cd cannibal_missionary

# install python requirements
pip3 install -r requirements.txt

# run the project
python3 game.py
```
